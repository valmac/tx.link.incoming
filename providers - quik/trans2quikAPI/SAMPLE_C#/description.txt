В командной строке можно указывать следующие параметры
test3.exe [тип транзакции] [имя файла] [интервал в милисекундах между отправкой транзакции]
тип транзакции: A-асинхронная, S-синхронная
имя файла - путь к файлу с транзакциями
интервал - целое число

В файле settings.txt в первой строке указывается каталог с запущенным клиентским местом

Пример
test3.exe S trr.in 4000
Программа подписывается на заявки и сделки по ВСЕМ инструментам
затем начинает с периодичностью в 4 секунды читать файл trr.in и отправлять из него синхронные транзакции

Данные по зявкам записываются в файл orders.log
Данные по сделкам записываются в файл trades.log
Результаты выполнения асинхронных транзакции записываются в async_trans.log
Результат выполения синхронных транзакции выводится на экран
